# Project Docs for AI Agents

This folder contains practical docs to help human and AI contributors work in this repository quickly and safely.

## Docs Map

- `docs/architecture.md`: System-level architecture and data flow.
- `docs/carfindr-app.md`: Next.js app structure (`carfindr/`) and key modules.
- `docs/search-notifier.md`: Go notification service (`searchNotifier/`) internals.
- `docs/database-and-data.md`: PostgreSQL schema, seed data, and shared model mapping.
- `docs/dev-setup-and-commands.md`: Local setup and common development commands.

## Scope Note

This is a living docs baseline. It is intentionally concise and optimized for onboarding agents first, then expanded over time as the code evolves.
